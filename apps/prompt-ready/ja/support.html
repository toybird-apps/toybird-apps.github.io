基本的な使い方

  1. Prompt Readyを開き、行いたい作業の種類を選びます。
  2. 元の文章、テーマ、背景情報を貼り付けるか、直接入力します。
  3. 必要に応じて、読み手、文体、出力言語、候補数、推奨案、リスク、情報源などの表示された設定を調整します。
  4. 画面右側に作成されたAI向け依頼文を確認します。
  5. 依頼文をコピーを押し、利用するAIサービスへ自分で貼り付けます。

よくある質問

入力した文章をAIサービスへ送信しますか？

いいえ。元の文章と作成した依頼文は、利用者自身がコピーして別のサービスへ貼り付けるまでMac内に留まります。

どのAIサービスで使えますか？

ChatGPT、Claude、Gemini、その他・汎用の形式を選択できます。作成した依頼文は、文章入力に対応した他のAIサービスへコピーして利用することもできます。

メニューバーアイコンでは何ができますか？

よく使う用途、おすすめ、お気に入り、ホーム画面、設定へ素早くアクセスできます。メインウインドウを閉じずに、別の用途へ切り替えられます。

お気に入りはいくつ保存できますか？

よく使う活用アイデアを最大3件まで保存できます。

クリップボードからの貼り付けはどのように動きますか？

貼り付け操作を選んだときだけ、クリップボードにある文章を読み取ります。クリップボード履歴は保存しません。

表示言語を変更できますか？

Prompt Readyの画面表示は、macOSでアプリに設定した言語に従います。AI向け依頼文の出力言語は、編集画面で個別に選択できます。元の文章に合わせる設定も利用できます。

対応するmacOSは？

macOS Ventura 13以降に対応します。

外部AIサービス

依頼文をコピーした後、次のWebサイトを開けます。文章の自動貼り付けや自動送信は行いません。

  • ChatGPT
  • Claude
  • Gemini

Prompt Readyは独立したアプリであり、各サービスの提供者が提供、推奨、または運営する公式アプリではありません。
