Prompt Ready

プライバシーポリシー

最終更新日：2026年6月13日

Prompt Readyは、利用者のプライバシーを尊重します。このポリシーでは、元の文章、作成した依頼文、クリップボード、端末内の設定、外部AIサービスへの任意のリンクの取り扱いを説明します。

Mac内での処理

入力した元の文章やテーマ、アプリが作成した依頼文は、Mac内で処理されます。Prompt Readyは、その内容をToybird Appsのサーバーや外部AIサービスへ自動送信しません。

クリップボード

貼り付け操作を選んだ場合、Prompt Readyはクリップボードにある文章を読み取り、編集画面へ入力します。コピー操作を選んだ場合、作成した依頼文をクリップボードへ書き込みます。

Prompt Readyは、クリップボード履歴を収集、送信、保存しません。

Mac内へ保存する設定

Prompt Readyは、選択中の利用先AI、初回案内の完了状態、起動時にホーム画面を開くか、お気に入りに登録した活用アイデアのID最大3件など、必要最低限の設定をMac内へ保存します。

入力した元の文章や作成した依頼文をToybird Appsのサーバーへ保存しません。

外部AIサービスへのリンク

利用者がリンクを選んだ場合、Prompt ReadyからChatGPT、Claude、GeminiのWebサイトを開けます。外部サービスへの自動ログイン、自動貼り付け、自動送信は行いません。

外部AIサービスを利用する場合、利用者が提供する情報には、そのサービスの利用規約とプライバシーポリシーが適用されます。

アカウント、広告、分析ツール、開発者サーバー

Prompt Readyは、アカウント登録やログインを必要としません。広告SDKや分析SDKを利用せず、元の文章や作成した依頼文を処理するためにToybird Appsのサーバーを使用しません。

第三者のサービス名

ChatGPT、Claude、Geminiの名称は、作成した依頼文の利用先を選択できるようにする目的でのみ記載しています。Prompt Readyは独立したアプリであり、各サービスの提供者が提供、推奨、または運営する公式アプリではありません。

ポリシーの更新

アプリの機能やデータの取り扱いが変わった場合、このポリシーを更新することがあります。更新日は、このページに表示します。

お問い合わせ
