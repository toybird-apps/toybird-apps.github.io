Pointer Cue

Show exactly where to look.
Pointer Cue helps you guide attention during software demos, presentations, and screen sharing.

Highlight the pointer instantly

Press and release the Command key to display a shrinking ring around the pointer and draw attention to the right spot.

Outline an area clearly

Hold Command and left-drag to draw a temporary rectangle without accidentally interacting with the app behind it.

Adjust colors and language

Choose the cue color that works best for your screen. Pointer Cue also supports multiple menu languages.

System Requirements

macOS Sequoia 15.6 or later. Designed for both Intel Macs and Apple silicon Macs.

Pointer Cue uses macOS Input Monitoring only to detect a standalone Command-key press and Command + left-drag gestures. It does not record, store, transmit, or analyze typed text, passwords, or input history.
